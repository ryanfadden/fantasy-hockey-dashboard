üöÄ Run the Full Dashboard Locally

To access your team data and recommendations:

  1. Clone the repository:
    git clone https://github.com/ryanfadden/fantasy-hockey-dashboard.git
  2. Navigate to the project:
    cd fantasy-hockey-dashboard
  3. Install dependencies:
    pip install -r requirements.txt
  4. Set up environment variables:
    cp env.example .env

    Then edit .env with your ESPN credentials and OpenAI API key

  5. Run the dashboard:
    python dashboard.py
  6. Open in browser: http://localhost:8050
